## Migration Summary

On **January 2, 2026**, the custom three-level architecture (User/Workspace/Repo)
was migrated to the official Claude Code architecture (User/Repo/Local).

### What Changed

| Before (Custom) | After (Official) | Action Taken |
|-----------------|------------------|--------------|
| ~/dev/.claude/agents/ | ~/.claude/agents/ or &lt;repo&gt;/.claude/agents/ | Distributed to user or repo level |
| ~/dev/.claude/skills/ | ~/.claude/skills/ | Moved to user level |
| ~/dev/.claude/commands/ | ~/.claude/skills/ | Converted to user-level skills |
| ~/dev/.claude/guides/ | ~/.claude/guides/ | Moved to user level |
| ~/dev/.claude/skill-templates/ | ~/dev/scripts/skill-templates/ | Moved to workspace scripts/ |
| ~/dev/.claude/reports/ | ~/dev/docs/archives/ | Archived as documentation |
